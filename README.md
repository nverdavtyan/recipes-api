
## Recipes Api
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

### Requirements

- Nest
- PostgreSQl - `config your db`

## Installation

```bash
$ Customize the `.env`
$ npm install

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
